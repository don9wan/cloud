# ProductCompass

개인 포트폴리오 웹사이트

## 🚀 기술 스택

- **Frontend**: React 18, TypeScript, Vite
- **Styling**: CSS Modules, Tailwind CSS
- **UI Components**: Radix UI, Lucide React
- **Animation**: GSAP, Framer Motion
- **State Management**: TanStack Query
- **Deployment**: Vercel

## 📦 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build

# 빌드 미리보기
npm run preview
```

## 🌐 배포

이 프로젝트는 Vercel에서 자동으로 배포됩니다.

## 📝 주요 기능

- 반응형 디자인
- 다크 모드 (고정)
- 부드러운 스크롤 애니메이션
- 프로젝트 갤러리
- 연락처 및 소셜 링크
- 이력서 다운로드 (준비 중)
